# ¡Bienvenidx a AdventMX 2025!

Este repo es simplemente el POC de la landing page del proyecto.

Realizado en Reflex, por lo que es puramente Python.

Si quieres levantarla por tu cuenta para apoyar a mejorarla puedes
hacer:

```bash
uv sync
source .venv/bin/activate
reflex init
reflex run
```

Esto abrirá el puerto 3000 en localhost para el frontend y el puerto 8000 en
el backend (aunque realmente éste no se ocupa).
